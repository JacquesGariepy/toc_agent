## Get started

1. Set up your Python environment

```
python -m venv env
```
#### For Linux and MacOS
```
source env/bin/activate
```
#### For Windows
```
env\Scripts\activate
```

2. Install dependencies

```
npm install
npm init -y
pip install -r requirements.txt 

```

3. Run system

```
npm run dev
cd src ; uvicorn api:app 
```
